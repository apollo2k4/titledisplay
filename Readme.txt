Copyright (C) 2009-2010 SEG Tech <me@gogogadgetscott.info>

PREFACE

This is a plugin for KeePass that allows the management of the main window's title
display.

REQUIREMENTS

This plugin requires KeePass 2.0x.

INSTALLING

Just copy TitleDisplay.dll to the same directory where KeePass.exe is located
and KeePass should automatically recognize and load the plugin.
