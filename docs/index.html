Title Display plugin for KeePass Password Safe
Plugin Author: GoGoGadgetScott, Plugin Language:

This is a plugin for KeePass that allows the management of the main window's title display.

Requires KeePass ≥ 2.42.

Copyright (C) 2009-2010 SEG Tech

Features:
» Controls Main Window Title.
» Controls tray icon tool tip.
» Show Product Name.
» Show Modify Date.
» Set Maximun Title Length.
» Plugin settings stored in KeePass Configuration file.
» Options for:
  » Database Name
  » DatabaseFile Name
  » Database File Path
  » Default User Name
  » Database Description
  » Database Root Group Name
  » Custom

Preface

This is a plugin for KeePass that allows the management of the main window title display.

Installation

  1. Simply copy the TitleDisplay.dll to the directory of KeePass.exe.
  2. Start KeePass

Configuration

A GUI is available for the configuration. To reach the configuration dialog, please click on menus:

Tools > Title Display...

There you'll be able to set the title display options.

Limitations

  • Multiple languages are not currently supported.
  • Plugin assembly dll is not signed.
